# agent-templates
Templates for coordinator-driven tetraphasic workflow agents, config-driven global-agent workflows, optional advanced-QA extensions, and related prompts.

Config-driven workflow templates:
- `workflow-config-template.md` defines the repository-local contract consumed by the global generic tetraphasic agents.
- `config-mode-kickoff-template.md` defines the repository-local kickoff prompt that invokes `@tetraphasic-coordinator`.
